# ProgressBar for Google Colab

![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)

```
from Progress import Bar

range_length = 1000
pb = Bar(range_length)

for i in range(range_length):
  pb.next()
```
